Lesson 1: Why HTML?

HTML هي الأساس اللي بيتبني عليه أي موقع على الإنترنت. بتستخدم علشان:

  • ترتيب وتنظيم عناصر الصفحة (عناوين، فقرات، صور...)
  • المتصفح بيقرأ كود HTML ويعرضه للمستخدم

HTML مش لغة برمجة، دي لغة وصف وهيكلة للمحتوى.

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>My First HTML Page</title>
</head>
<body>

  <!-- This is the main heading -->
  <h1>Welcome to My Website</h1>

  <!-- This is a paragraph introducing the site -->
  <p>Hello! I'm learning how to build websites using HTML.</p>

</body>
</html>

Lesson 2: How Browser Reads HTML?

المتصفح مش بيعرض كود HTML نفسه، هو بيقرأه ويفهم الوسوم، ويعرض المحتوى اللي جواه.

مثلًا: لو كتبت وسم الفقرة <p>، المتصفح هيعرض النص فقط.

<!-- This is a paragraph -->
<p>I love programming!</p>

Lesson 3: First HTML Project

علشان تبدأ أول مشروع:

  1. افتح محرر كود زي VS Code
  2. اعمل ملف جديد باسم index.html
  3. اكتب الهيكل الأساسي للصفحة

الهيكل الأساسي بيتكوّن من:

  • الرأس (head): فيه معلومات مخفية
  • الجسم (body): فيه كل المحتوى اللي المستخدم بيشوفه
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>Welcome</title>
</head>
<body>
  <h1>This is my first web page</h1>
  <p>I am learning HTML to create websites.</p>
</body>
</html>

Lesson 4: Page Structure

أي صفحة HTML لازم يكون فيها عناصر أساسية، وكل عنصر ليه وظيفة:

العنصر (الوسم)المعنى (وظيفته)
<!DOCTYPE html>بيعرف المتصفح إن الملف ده مكتوب بلغة HTML5
<html>الجذر الأساسي للصفحة، وكل العناصر بتتكتب جواه
<head>فيه معلومات بتفيد المتصفح (زي العنوان والروابط)
<title>بيظهر فوق في تبويب المتصفح
<body>الجزء اللي بيشوفه المستخدم: عناوين، فقرات، صور، إلخ
 
<!DOCTYPE html>
<html>
  <head>
    <title>Page Structure</title>
  </head>
  <body>
    <h1>Page Content</h1>
    <p>This part is inside the body.</p>
  </body>
</html>

Lesson 5: Comments in HTML

التعليقات بتستخدم لكتابة ملاحظات جوه الكود، المتصفح بيتجاهلها ومش بيعرضها للمستخدم.

<!-- This is a comment and will not appear on the page -->

<h2>News</h2>
<!-- The section below is for latest news -->
<p>Elzero released a new HTML lesson!</p>

Lesson 6: Headings

فيه 6 مستويات من العناوين من <h1> (أكبر) إلى <h6> (أصغر). بنستخدمها لترتيب المحتوى.

<h1>This is Heading 1</h1>
<h2>This is Heading 2</h2>
<h3>This is Heading 3</h3>
<h4>This is Heading 4</h4>
<h5>This is Heading 5</h5>
<h6>This is Heading 6</h6>

Lesson 7: Paragraphs

بستخدم وسم <p> علشان تكتب فقرة. كل فقرة لازم تكون محاطة بالوسم ده.

<p>This is a paragraph in HTML.</p>

Lesson 9: HTML Attributes

السمات (Attributes) بتدي معلومات إضافية للعنصر زي: id، class، title، href، src.

<p title="This is a tooltip">Hover over me!</p>
<a href="https://elzero.org">Visit Elzero</a>

Lesson 10: Lists

Unordered List: <ul> | Ordered List: <ol> | كل عنصر: <li> 

<ul>
  <li>HTML</li>
  <li>CSS</li>
</ul>

<ol>
  <li>Step 1</li>
  <li>Step 2</li>
</ol>

Lesson 11: Links

بستخدم وسم <a> علشان تعمل رابط. السمة href بتحدد مكان الرابط.

<a href="https://elzero.org" target="_blank">Go to Elzero</a>

Lesson 12: Images

بستخدم وسم <img> وبتحدد مصدر الصورة بـ src والنص البديل بـ alt.

<img src="image.jpg" alt="Description of image" width="300">

Lesson 13: Tables

الجداول بتتكون من <table> وفيها <tr> صفوف، <td> خلايا، و <th> رؤوس.

<table border="1">
  <tr><th>Name</th><th>Age</th></tr>
  <tr><td>Ali</td><td>25</td></tr>
</table>

Lesson 14: Special Characters

بعض الرموز زي < و > لازم نكتبها بطريقة خاصة عشان المتصفح ما يفسرشها ككود.

<p>Use &lt; and &gt; to show angle brackets</p>
<p>Copyright: &copy; 2025</p>

Lesson 15: Text Formatting

وسوم لتنسيق النصوص مثل <b> (غامق)، <i> (مائل)، <mark> (تحديد)، إلخ.

<p>This is <b>bold</b>, <i>italic</i>, and <mark>highlighted</mark></p>

Lesson 16: div and span

<div> عنصر بلوكي لتقسيم الصفحة، و <span> عنصر خطي لتنسيق جزء من النص. 

<div>Block section here</div>
<p>This is a <span style="color:red">red</span> word.</p>

Lesson 17: Empty Elements

عناصر ملهاش وسم إغلاق زي <br> و <hr> و <img>.

<p>Line 1<br>Line 2</p>
<hr>
<img src="https://via.placeholder.com/150" alt="Sample Image">

Lesson 18: Button Element

بستخدم وسم <button> لإنشاء زر ضغط.

<button>Click Me</button>

Lesson 19: HTML Comments

بنستخدم <!-- Comment --> علشان نكتب ملاحظات لا تظهر للمستخدم في المتصفح.

<!-- This is a comment and will not appear -->
<h2>News</h2>
<p>Elzero released a new HTML lesson!</p>

Lesson 20: Full Review

صفحة مراجعة بتجمع كل اللي اتعلمناه من عناصر أساسية في HTML.

<!DOCTYPE html>
<html>
  <head>
    <title>Review Page</title>
  </head>
  <body>
    <h1>Welcome</h1>
    <p>This is a <b>review</b> of HTML basics.</p>
    <hr>
    <ul><li>HTML</li></ul>
    <a href="#">Link</a>
    <br>
    <img src="img.jpg" alt="Sample">
  </body>
</html>

Lesson 21: Nested Elements

تقدر تكتب وسوم داخل وسوم تانية، زي إنك تكتب <b> أو <i> جوه <p>.

<p>This is a <b>bold</b> and <i>italic</i> word.</p>

Lesson 22: Code Indentation

من الأفضل تنظم الكود بمسافات وتعمل مسافات بادئة (Indentation) علشان الكود يكون مقروء وسهل الفهم.

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>

Lesson 23: Internal & External Links

الرابط الخارجي بيربطك بصفحة خارج موقعك، والداخلي بيربطك بصفحة داخل نفس الموقع.

<a href="https://google.com" target="_blank">External Link</a>
<br>
<a href="about.html">Internal Link</a>

Lesson 24: Mail & Phone Links

ممكن تعمل رابط يفتح برنامج البريد لإرسال رسالة، أو يتصل برقم تليفون مباشرة عند الضغط.

<a href="mailto:example@email.com">Send Email</a>
<br>
<a href="tel:+201234567890">Call Now</a>

Lesson 25: Image as Link

ممكن تحط صورة جوا رابط، بحيث لما المستخدم يضغط على الصورة، تفتح صفحة معينة.

<a href="https://elzero.org">
  <img src="logo.png" alt="Elzero Logo">
</a>

Lesson 26: Button Inside Link

ممكن تحط زر داخل رابط علشان تعطي شكل زر لرابط، لكن لازم تستخدمه بحذر علشان يكون متوافق مع المعايير.

<a href="https://elzero.org">
  <button>Visit Elzero</button>
</a>

Lesson 27: Image Width & Height

تقدر تتحكم في عرض وارتفاع الصورة باستخدام السمات width وheight.

<img src="photo.jpg" width="200" height="150" alt="A photo">

Lesson 28: Images from Internet

تقدر تعرض صورة مباشرة من الإنترنت باستخدام رابط مباشر للصورة في السمة src.

<img src="https://via.placeholder.com/150" alt="Placeholder Image">

Lesson 29: Image Alt Text

النص البديل alt بيظهر لما الصورة متقدرش تتحمّل، وكمان بيساعد الأشخاص اللي بيستخدموا قارئات الشاشة.

<img src="invalid.jpg" alt="Image not found">

Lesson 30: HTML Entities

بعض الرموز في HTML لها استخدامات خاصة زي < و >، فلازم نكتبها باستخدام كيانات HTML علشان تظهر كرموز عادية.

<p>Trademark: &trade; | Registered: &reg;</p>
<p>Use &lt; and &gt; to show angle brackets</p>

Lesson 31: HTML Video

تقدر تستخدم وسم <video> لعرض فيديو داخل الصفحة، مع خصائص زي controls وautoplay وloop.

<video width="320" height="240" controls>
  <source src="movie.mp4" type="video/mp4">
  Your browser does not support the video tag.
</video>

Lesson 32: HTML Audio

وسم <audio> بيتيح لك تشغيل ملفات صوتية على موقعك، وتقدر تضيف له خصائص زي controls وautoplay.

<audio controls>
  <source src="sound.mp3" type="audio/mpeg">
  Your browser does not support the audio element.
</audio>

Lesson 33: iframe

وسم <iframe> بيستخدم لعرض محتوى خارجي داخل صفحتك زي صفحة ويب تانية أو فيديو من YouTube.

<iframe src="https://elzero.org" width="400" height="300"></iframe>

Lesson 34: HTML Forms

بنستخدم وسم <form> لجمع بيانات من المستخدم زي الاسم والإيميل، ووسوم <input> و<label> وغيرها لعرض الحقول.

<form>
  <label>Name:</label>
  <input type="text" name="username">
  <br>
  <input type="submit" value="Send">
</form>

Lesson 35: Input Types

وسم <input> ليه أنواع كتير. كل نوع بيحدد طريقة إدخال مختلفة زي: text، email، password، number، إلخ.

<input type="email" name="useremail" placeholder="Enter your email">

Lesson 36: Radio & Checkbox

نستخدم type="radio" لاختيار عنصر واحد من مجموعة، وtype="checkbox" لاختيار عناصر متعددة. 

<input type="radio" name="gender" value="male"> Male
<input type="radio" name="gender" value="female"> Female
<br>
<input type="checkbox" name="skills" value="html"> HTML
<input type="checkbox" name="skills" value="css"> CSS

Lesson 37: Select Menu

نستخدم وسم <select> لعرض قائمة منسدلة، ووسم <option> لتمثيل العناصر داخلها.

<select name="country">
  <option value="eg">Egypt</option>
  <option value="sa">Saudi Arabia</option>
</select>

Lesson 38: Textarea

وسم <textarea> بيُستخدم لما تحتاج من المستخدم يكتب نص طويل زي ملاحظات أو رسالة.

<textarea name="message" rows="4" cols="30"></textarea>

Lesson 39: HTML5 Semantic Tags

HTML5 قدمت عناصر دلالية (Semantic) بتوضح وظيفة كل جزء في الصفحة، وبتساعد محركات البحث والقارئات. أهم الوسوم: <header>, <footer>, <article>, <section>, <nav>, إلخ. 

<header>
  <h1>Site Title</h1>
</header>
<section>
  <article>
    <h2>Post Title</h2>
    <p>Content...</p>
  </article>
</section>
<footer>
  <p>All rights reserved.</p>
</footer>

Lesson 40: End of Part One

مبروك! وصلت لنهاية الجزء الأول من تعلم HTML. دلوقتي بقي عندك أساس قوي في هيكلة صفحات الويب، وده هيساعدك تكمل باقي الرحلة في تصميم المواقع. 

<h1>Thanks for learning!</h1>
<p>You completed the HTML beginner series.</p>
